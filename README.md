# Tamirci İbo

Etimesgut, Ankara'da bulunan **Tamirci İbo** için hazırlanmış modern, mobil öncelikli tek sayfa web sitesi.

## Dosyalar

- `index.html` - Ana sayfa
- `styles.css` - Tasarım ve animasyonlar
- `script.js` - Yıl bilgisi ve scroll animasyonları
- `pictures/` - Yerel görseller ve logo
- `render.yaml` - Render static site deploy ayarı

## Render Deploy

1. Bu projeyi GitHub repository olarak yükleyin.
2. Render panelinde **New +** > **Blueprint** seçin.
3. GitHub repository'yi bağlayın.
4. Render, `render.yaml` dosyasını okuyarak static site ayarlarını otomatik kullanır.

Alternatif olarak Render'da **New +** > **Static Site** seçip:

- Build Command: boş bırakın
- Publish Directory: `.`

olarak ayarlayabilirsiniz.
